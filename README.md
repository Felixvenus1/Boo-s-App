# Boo Virtual Pet

A simple virtual pet game built with HTML, CSS, and JavaScript.

## Run locally

### Download as ZIP (no Git required)

1. Open the repository page: `https://github.com/Felixvenus1/Boo-s-App`.
2. Click the green `Code` button near the top-right of the file list.
3. Click `Download ZIP`.
4. Find the downloaded ZIP file (usually in your `Downloads` folder).
5. Right-click the ZIP and choose `Extract All...`, then pick a folder.
6. Open the extracted folder and double-click `booapp.html`.

The app should open in your default browser.

Or clone with Git:

```bash
git clone https://github.com/Felixvenus1/Boo-s-App.git
cd Boo-s-App
```

Then open `booapp.html`.

## Optional: run a local server

If you have Python installed:

```bash
python -m http.server 8000
```

Open `http://localhost:8000`.
