# Boo Virtual Pet

A simple virtual pet game built with HTML, CSS, and JavaScript.

## Run locally

1. Download this repository as ZIP and extract it.
2. Open `booapp.html` in any modern browser.

Or clone with Git:

```bash
git clone https://github.com/Felixvenus1/Boo-s-App.git
cd Boo-s-App
```

Then open `booapp.html`.

## Optional: run a local server

If you have Python installed:

```bash
python -m http.server 8000
```

Open `http://localhost:8000`.
